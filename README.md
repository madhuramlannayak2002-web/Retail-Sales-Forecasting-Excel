# Retail Sales Forecasting Dashboard (Excel)

This project analyzes retail sales data and builds a forecasting dashboard using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Forecast Sheet
- Dashboard Design

## Key Metrics
Total Sales: ₹456,000  
Total Transactions: 1000  
Average Order Value: ₹456  

## Features
- Monthly sales trend analysis
- Product category performance
- Customer purchase analysis by gender
- Sales forecasting for upcoming months
- Interactive Excel dashboard
